# 婚活タイプ診断

🔗 **https://yuu2811.github.io/KONKATSU/**

---

愛着理論（Bartholomew & Horowitz, 1991）に基づく心理学的診断で、あなたの婚活スタイルを分析します。

## 特徴

- **20問の診断テスト** — 2軸（開放性 × 主体性）で婚活タイプを判定
- **8つの診断タイプ** — 強み・改善ポイント・具体的アドバイスを提供
- **統計データ引用** — 国立社会保障・人口問題研究所、リクルートブライダル総研、厚生労働省等の公的データに基づくアドバイス
- **SNSシェア** — X (Twitter)・LINE・リンクコピー対応

## 診断タイプ一覧

| タイプ | 象限 |
|---|---|
| ☀️ 太陽のナビゲーター | 高開放 × 高主体 |
| 🌈 あたたかい架け橋タイプ | 高開放 × 高主体（中） |
| 🔥 情熱のチャレンジャー | 高開放 × 低主体 |
| 🌙 やさしいロマンチスト | 高開放 × 低主体（中） |
| ♟️ 静かなるストラテジスト | 低開放 × 高主体 |
| 🧱 堅実なビルダー | 低開放 × 高主体（中） |
| 🔍 慎重なアナリスト | 低開放 × 低主体 |
| ☁️ やさしいドリーマー | 低開放 × 低主体（中） |

## 技術構成
